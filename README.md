# Retail Sales Analysis using SQL

## Project Overview

This project focuses on analyzing retail sales data using SQL to uncover valuable business insights. The analysis includes data cleaning, exploratory data analysis (EDA), and solving key business questions related to customer behavior, product performance, and sales trends.

The project demonstrates practical SQL skills such as:

* Data Cleaning
* Data Exploration
* Aggregation Functions
* Window Functions
* Common Table Expressions (CTEs)
* Date and Time Analysis
* Business Intelligence Reporting

---

## Database Setup

Create the database:

```sql
CREATE DATABASE retail_sales;
```

Select the database:

```sql
USE retail_sales;
```

---

## Dataset Information

The dataset contains retail transaction records with the following attributes:

| Column Name     | Description                   |
| --------------- | ----------------------------- |
| transactions_id | Unique transaction identifier |
| sale_date       | Date of sale                  |
| sale_time       | Time of sale                  |
| customer_id     | Unique customer identifier    |
| gender          | Customer gender               |
| age             | Customer age                  |
| category        | Product category              |
| quantity        | Quantity purchased            |
| price_per_unit  | Price per unit                |
| cogs            | Cost of goods sold            |
| total_sale      | Total transaction value       |

## Business Problems Solved

### 1. Sales Made on a Specific Date

Retrieve all sales made on:

```text
2022-11-05
```

### 2. High-Volume Clothing Orders

Find all Clothing category transactions where:

* Quantity ≥ 4
* Sale occurred during November 2022

### 3. Total Revenue by Category

Calculate total sales generated by each product category.

### 4. Average Age of Beauty Customers

Determine the average age of customers purchasing products from the Beauty category.

### 5. High-Value Transactions

Identify all transactions with total sales greater than 1000.

### 6. Gender-wise Transactions per Category

Analyze transaction counts grouped by category and gender.

### 7. Best Selling Month of Each Year

Find the month with the highest average sales for every year using SQL window functions.

### 8. Top 5 Customers by Spending

Identify customers who contributed the highest total sales revenue.

### 9. Unique Customers per Category

Calculate the number of distinct customers purchasing from each category.

### 10. Sales Shift Analysis

Categorize transactions into:

* Morning (Before 12 PM)
* Afternoon (12 PM to 5 PM)
* Evening (After 5 PM)

Determine the number of transactions occurring during each shift.
